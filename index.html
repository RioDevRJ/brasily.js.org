Documentação da Brasily JS

Brasily JS é uma biblioteca JavaScript leve e intuitiva, que une a elegância das cores vibrantes do Brasil a uma abordagem minimalista para desenvolvimento web. Ela oferece um conjunto robusto de funções para manipulação de DOM, validação de dados, requisições AJAX e muito mais – tornando o desenvolvimento de aplicações web rápido e eficiente.

Funções de Input

Retorna o valor do primeiro elemento encontrado pelo seletor informado.

  • Parâmetros: seletor (string) – Seletor CSS.
  • Retorno: Valor (string) do primeiro elemento encontrado.
const valor = br.inputVal('#meuInput');

Retorna um array com os valores de todos os elementos encontrados pelo seletor.

  • Parâmetros: seletor (string).
  • Retorno: Array de valores (Array).
const valores = br.inputValues('.inputs');

Limpa os valores dos elementos do tipo input ou select encontrados.

  • Parâmetros: seletor (string, opcional); padrão: 'input,select'.
  • Retorno: Nenhum.
br.inputClean();
br.inputClean('.meus-campos');

Desativa o autocomplete de todos os elementos input se o parâmetro for false.

  • Parâmetros: v (boolean, opcional); padrão: false.
  • Retorno: Nenhum.
br.inputAutoComplete();
br.inputAutoComplete(true);

Manipulação de Elementos

Retorna o primeiro elemento encontrado utilizando o seletor informado.

  • Parâmetros: seletor (string).
  • Retorno: Elemento (HTMLElement).
const el = br.element('.minha-div');

Retorna um array com os valores de todos os elementos encontrados.

  • Parâmetros: seletor (string).
  • Retorno: Array de valores.
const todosValores = br.elements('.inputs');

Substitui o innerHTML dos elementos encontrados.

  • Parâmetros: seletor (string) e conteudo (string).
  • Retorno: Nenhum.
br.addHTML('.minha-div', '<p>Olá Mundo</p>');

Remove o conteúdo (innerHTML) dos elementos encontrados.

  • Parâmetros: seletor (string).
  • Retorno: Nenhum.
br.removeHTML('.minha-div');

Retorna o conteúdo (innerHTML) do primeiro elemento encontrado.

  • Parâmetros: seletor (string).
  • Retorno: string.
const conteudo = br.getHTML('.minha-div');

Acrescenta o conteúdo informado ao final do innerHTML dos elementos encontrados.

  • Parâmetros: seletor (string) e conteudo (string).
  • Retorno: Nenhum.
br.addNextHTML('.minha-div', '<p>Texto Adicional</p>');

Retorna uma cópia profunda do primeiro elemento encontrado.

  • Parâmetros: seletor (string).
  • Retorno: Elemento clonado (HTMLElement).
const copia = br.clone('.minha-div');

Retorna um objeto contendo os parâmetros da URL atual.

  • Parâmetros: Nenhum.
  • Retorno: Objeto ({ chave: valor, ... }).
const params = br.getUrl();

Manipulação de Classes

Adiciona a classe CSS especificada a todos os elementos encontrados.

  • Parâmetros: seletor (string) e classe (string).
  • Retorno: Nenhum.
br.addClass('.minha-div', 'ativa');

Remove a classe CSS especificada de todos os elementos encontrados.

  • Parâmetros: seletor (string) e classe (string).
  • Retorno: Nenhum.
br.removeClass('.minha-div', 'ativa');

Manipulação de Atributos

Retorna o valor do atributo especificado do primeiro elemento encontrado.

  • Parâmetros: seletor (string) e atributo (string).
  • Retorno: Valor do atributo (string).
const alt = br.getAtributte('img', 'alt');

Remove o atributo especificado de todos os elementos encontrados.

  • Parâmetros: seletor (string) e atributo (string).
  • Retorno: Nenhum.
br.removeAtributte('.minha-div', 'data-info');

Define o atributo informado com o valor especificado para todos os elementos encontrados.

  • Parâmetros: seletor (string), atributo (string) e valor (string).
  • Retorno: Nenhum.
br.addAtributte('.minha-div', 'title', 'Minha Div');

Manipulação de Eventos

Adiciona um listener para o evento especificado a todos os elementos encontrados.

  • Parâmetros: seletor (string), evento (string) e callback (function).
  • Retorno: Nenhum.
br.event('.botao', 'click', () => { alert('Clicou!'); });

Remove o listener do evento especificado dos elementos encontrados.

  • Parâmetros: seletor (string), evento (string) e callback (function).
  • Retorno: Nenhum.
br.removeEvent('.botao', 'click', minhaFuncao);

Oculta os elementos encontrados definindo seu display como "none".

  • Parâmetros: seletor (string).
  • Retorno: Nenhum.
br.hide('.minha-div');

Exibe os elementos encontrados definindo seu display como "block".

  • Parâmetros: seletor (string).
  • Retorno: Nenhum.
br.show('.minha-div');

Alterna entre exibir e ocultar os elementos encontrados.

  • Parâmetros: seletor (string).
  • Retorno: Nenhum.
br.toggle('.minha-div');

Busca de Elementos

Busca um elemento interno a partir do primeiro elemento encontrado pelo seletor e aplica um seletor de busca.

  • Parâmetros: seletor (string) e busca (string – seletor interno).
  • Retorno: Primeiro elemento que corresponder à busca.
const filho = br.find('.pai', '.filho');

Requisições

Realiza uma requisição GET para a URL informada e retorna os dados JSON através de um callback.

  • Parâmetros: url (string) e callback (function).
  • Retorno: Nenhum (os dados são passados para o callback).

br.getJson('https://api.exemplo.com/dados', (data) => {
  console.log(data);
});

Realiza uma requisição POST enviando dados JSON e retorna a resposta via callback.

  • Parâmetros:
    • url (string)
    • cabecalhos (object)
    • dados (object)
    • callback (function)
  • Retorno: Nenhum (os dados são passados para o callback).

br.postJson('https://api.exemplo.com/salvar', 
  { 'Authorization': 'Bearer token' },
  { nome: 'Teste' },
  (data) => {
    console.log(data);
});

Validações de Dados

Verifica se o valor informado é uma string não nula.

  • Parâmetros: valor (qualquer).
  • Retorno: Boolean.
const resultado = br.isString('teste');

Verifica se o valor é um número inteiro.

  • Parâmetros: valor (number).
  • Retorno: Boolean.
const inteiro = br.isInt(10);

Verifica se o valor é um número float.

  • Parâmetros: valor (number).
  • Retorno: Boolean.
const flutuante = br.isFloat(10.5);

Verifica se o valor é do tipo objeto (JSON).

  • Parâmetros: valor (qualquer).
  • Retorno: Boolean.
const ehJson = br.isJson({ chave: 'valor' });

Verifica se o valor é do tipo boolean.

  • Parâmetros: valor (qualquer).
  • Retorno: Boolean.
const ehBool = br.isBool(true);

Verifica se o valor não é nulo/undefined.

  • Parâmetros: valor (qualquer).
  • Retorno: Boolean.

const naoNulo = br.isNull('teste');
const definido = br.isUndefined('teste');

Funções para verificar:

  • isObject(valor) – se é um objeto.
  • isDocument(valor) – se é uma instância de Document.
  • isDoble(valor) – se é um número (double).
  • isFunction(valor) – se é uma função.
  • isElement(valor) – se é um elemento DOM.
  • isEvent(valor) – se é um evento.
  • isNodeList(valor) – se é um NodeList.
  • isHTMLCollection(valor) – se é uma HTMLCollection.
  • Funções específicas para elementos HTML, como isHTMLInputElement, etc.

  • isEmail(valor) – valida e-mail.
  • isCep(valor) – valida CEP (XXXXX-XXX).
  • isCpf(valor) – valida CPF (XXX.XXX.XXX-XX).
  • isCnpj(valor) – valida CNPJ (XX.XXX.XXX/XXXX-XX).
  • isTel(valor) – valida telefone ((XX) XXXX-XXXX ou (XX) XXXXX-XXXX).
  • isDate(valor) – valida data (dd/mm/yyyy).
  • isHr(valor) – valida hora (hh:mm).
  • isDateHr(valor) – valida data e hora (dd/mm/yyyy hh:mm).
  • isUrl(valor) – valida URL.

Valida números de CPF e CNPJ e testa se uma URL é válida.

  • Parâmetros: valor (string).
  • Retorno: Boolean.

const cpfValido = br.testCpf('123.456.789-09');
const cnpjValido = br.testCnpj('12.345.678/0001-95');
const urlValida = br.testUrl('https://exemplo.com');

Manipulação de Dados

Converte uma string JSON em um objeto.

  • Parâmetros: valor (string JSON).
  • Retorno: Objeto JavaScript.
const obj = br.parseJson('{"chave": "valor"}');

Converte um valor em uma string JSON.

  • Parâmetros: valor (qualquer).
  • Retorno: string.
const jsonStr = br.parseString({ chave: "valor" });

Converte o valor para um número float.

  • Parâmetros: valor (string ou number).
  • Retorno: Número (float).
const num = br.parseFloat("10.5");

Converte o valor para um número inteiro.

  • Parâmetros: valor (string ou number).
  • Retorno: Número (int).
const inteiro = br.parseInt("10");

Converte o valor para boolean.

  • Parâmetros: valor (qualquer).
  • Retorno: Boolean.
const boolVal = br.parseBool(1);

Converte o valor para um array utilizando Array.from().

  • Parâmetros: valor (iterable ou array-like).
  • Retorno: Array.
const arr = br.parseArray('texto');

Converte o valor em um objeto utilizando Object.assign().

  • Parâmetros: valor (qualquer).
  • Retorno: Objeto.
const obj = br.parseObject({ a: 1 });

Converte o valor para um objeto Date.

  • Parâmetros: valor (string ou number).
  • Retorno: Objeto Date.
const data = br.parseDate("2025-02-12");

Converte o valor para um número (double). (Nota: pode ser uma variação de "parseDouble")

  • Parâmetros: valor (string ou number).
  • Retorno: Número.
const numDouble = br.parseDoble("10.5");

Converte uma string em uma função.

  • Parâmetros: valor (string – código da função).
  • Retorno: Função.
const func = br.parseFunction("return 2+2");

Cria e retorna um novo elemento DOM com a tag especificada.

  • Parâmetros: valor (string – nome da tag, ex.: 'div').
  • Retorno: Elemento DOM.
const novaDiv = br.parseElement('div');

Function Builder

Retorna uma nova função que valida os tipos dos argumentos e do retorno conforme especificados.

  • Parâmetros:
    • callback (function) – função a ser executada.
    • returnType (string) – tipo esperado de retorno (ex.: 'string', 'int', etc.).
    • paramTypes (Array) – tipos esperados para os argumentos.
    • debug (boolean, opcional) – ativa mensagens de erro detalhadas.
  • Retorno: Função validada.

const funcValida = br.function(
  (a, b) => a + b,
  'int',
  ['int', 'int'],
  true
);
const resultado = funcValida(10, 20); // Retorna 30

API Request

Realiza uma requisição assíncrona utilizando fetch e retorna uma Promise com um array contendo o status e os dados.

  • Parâmetros:
    • url (string) – URL para a requisição.
    • method (string, opcional) – método HTTP (padrão: 'GET').
    • headers (object, opcional) – cabeçalhos adicionais.
    • body (object, opcional) – dados a serem enviados (para métodos diferentes de GET).
    • callback (function, opcional) – função que recebe (status, data).
  • Retorno: Promise que resolve para um array [status, data].

(async () => {
  const [status, data] = await br.api('https://api.exemplo.com', 'GET');
  console.log(status, data);
})();
